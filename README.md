https://chat.z.ai/ 智谱ai国际站，功能多，可选模型<br>
https://ai1.yjie.fun/#/chat Token限制，模型较旧，偶尔不稳定<br>
https://vwx2ic.aitianhu1.top/#/chat/ 基于NextChat<br>

https://svip.freeuse.top/ 镜像站，支持ChatGPT、Claude等的Team（或高级）账号，免费的只有ChatGPT，且受限，每日10次。只有一天的免费权限，之后都要付费，有轻微改装（增加会员入口）。<br>

https://www.chatgot.io/ 不错，但需要登录<br>
https://chatmoe.cn/ 国内小站，ui一般（有点简陋），GLM4.5<br>

https://chat3.leadscloud.xyz 真实可用，基于NextChat，体验不错，没有让输登录码。有很多无需登录的免费的模型<br>
https://cgs2.skybyte.me/ 有广告，普通<br>
https://xsimplechat.com/ 还不错，模型比较多<br>

https://chat.aihuoya.com 声称gpt中文版，综合体验一般，UI也没那么像ChatGPT<br>
https://easychat.top不付费可以用Claude免费账号，体验与外国的Claude一致，还不错。提供多个账号，经过轻微改装（将对话记录中的所有对话隐藏起来）。限制：未注册用户和普通用户是5小时10次，会员与所选择的套餐有关。对话不会串账号，可无缝切换（Chats页面单独查看）<br>

https://aizex.net/ 可以体验免费号，采用精心重构的1:1还原ChatGPT的UI，免费号有次数限制且不可恢复。完全改装<br>
https://mygpt.bid 有广告，一般<br>
